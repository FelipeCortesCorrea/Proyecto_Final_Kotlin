# Proyecto_Final_Kotlin
Este proyecto de kotlin fue basado en un proyecto de flutter pero esta vez estamos intentando hacerle lo mismo que flutter, pero toda via no esta conectado a ninguna base de datos, toda via esta en poceso su carrito de compras y sus apis 
## Contiene 
- Un SplashScreen
- Un iniciar Sesion
- Una pagina principal con sus respectivas Cards
- Un drawer
- Tiene diferentes paginas de categorias 
- Esta Conectado a GoogleMaps
- Tiene Canal de notificaciones
- Pagina de noticias
